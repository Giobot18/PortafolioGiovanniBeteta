CONOCIMIENTOS

HTML5

En este lenguaje de modelación comprendo las etiquetas para manejar una web semántica, la orientación a buscadores, APIS, Canvas, formularios etc.

CSS3

Con este lenguaje de estilos simplifico el mayor cumplimiento de estándares en el diseño de páginas, así como la compatibilidad con diversos navegadores, adaptando la presentación del contenido a características del dispositivo.

JAVA SCRIPT

Utilizo este lenguaje de interpretación para hacer animación de los elementos de una página, hago uso de la librería de jQuery que nos permite simplificar la manera de interactuar con la página.

BOOTSTRAP

Este Framework por su sintaxis que es muy fácil de usar y que nos brinda una mejor experiencia de usabilidad y diseño es el que utilizo comúnmente como estructura de una página, con el fin de dar siempre una mejor experiencia al usuario

UI/UX

Comprendo y Describo la experiencia e interfaz del usuario diseñando páginas útiles, usables y deseables, haciendo que el usuario se sienta satisfecho y se logre el objetivo de un negocio

Otros conocimientos

Reparación de PC

Tengo experiencia en reparación de computadoras tanto Hardware como Software; backups, recuperación de datos, formateo, mantenimientos preventivos, conocimientos de redes WAN y LAN.

REDES

Comprendo y describo los dispositivos y servicios utilizados para admitir comunicaciones en redes y en internet, capas y protocolos en redes de datos, la importancia de los esquemas de direccionamiento y de nomenclatura en diversas capas de las redes de datos en entornos IPv4 e IPv6, diseñar, calcular y aplicar mascaras de subred y direcciones para cumplir con determinados requisitos en redes IPv4 y IPv6.

Diseño

Aprendí a utilizar por cuenta propia Photoshop para la maquetación y diseño de una página web, también realice el diseño de logotipos para negocios propios con esta herramienta.
